# WebGado

Sistema gerenciador de animais de corte.

## Algumas Funcionalidades:

* Múltiplos Usuários

* Gerenciamento de:
    * Fazendas
    * Lotes de Animais
    * Peso e Tamanho
    * Vacinação
    * Medicamentos

* Dashboard (Gráficos da Situação Atual)

* Filtros de Pesquisas
